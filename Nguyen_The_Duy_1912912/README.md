# Bài tập lớn 2 - CSDL

## How to install
```
git clone https://github.com/fxminhquan/btl2-csdl.git
```
## Install necessary packages
### Install packages of client
```
cd client
npm install
```
### Install packages of backend
```
cd backend
npm install
```
## How to run
### Run backend
```
cd backend
npm start
```
### Run client
```
cd client
npm start
```
